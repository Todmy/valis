# Quickstart: Web Project Member Management

## Invite a Team Member

1. Open `valis.krukit.co` → sign in
2. Click a project name on the dashboard
3. Enter teammate's email in the invite form
4. Click "Invite"
5. Teammate receives an email with a login link
6. When they log in, the project appears in their dashboard

## Remove a Member

1. Open project detail page
2. Click "Remove" next to the member's name
3. Confirm in the dialog
4. Member is removed from this project (not from the org)

## Permissions

- **Project admins**: Can invite and remove members
- **Project members**: Can view the member list but cannot invite or remove
